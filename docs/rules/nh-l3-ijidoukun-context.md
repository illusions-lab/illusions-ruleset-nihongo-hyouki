# nh-l3-ijidoukun-context — 異字同訓の文脈判別（同音語の適切な漢字選択）

## 意図

「異字同訓」とは同じ訓読みを持ちながら異なる漢字が対応する語のことであり、常用漢字表・文化審議会の「"異字同訓"の漢字の使い分け例」（2014年）に使い分けの原則が示されている。
例えば「あう」は「会う」（人と顔を合わせる）・「遭う」（思わぬ出来事に出くわす）・「合う」（一致する/調和する）と使い分けが必要で、選択は文の意味的文脈に依存する。
この判断は形態素解析だけでは解決できず、LLM による文脈理解が必要。

## 書籍根拠

日本語表記（日本エディタースクール）第3章「漢字の用い方」§2 漢字の使用範囲

常用漢字表の音訓選定基準に「異字同訓はなるべく避けるが、漢字の使い分けのできるものは取り上げる」
という考え方が含まれており、書籍はその具体的な使い分け資料として
「異字同訓」の漢字の使い分け例（2014年 文化審議会国語分科会報告）を参照している。

書籍では「会う」「遭う」「合う」のような同訓異字の使い分けの考え方が紹介されている:
- 会う: 人と対面する場面に使う
- 遭う: 思わぬ出来事や好ましくない事態に遭遇する場面に使う
- 合う: 一致・調和・相互の場面に使う

## 動作原理

**LLM への入力**: 該当文および前後の文脈（段落単位が望ましい）

**判断基準**:
1. 文中に異字同訓の候補語（かな書きのまま、または誤字の漢字）が出現しているか検出する。
2. 文の意味的文脈（動作の対象・状況・意図）から正しい漢字を選択する。
3. 使用されている漢字が文脈と一致しない場合に逸脱として報告する。

**対象語の例** (文化審議会「使い分け例」より):
- あう: 会う（対面）/ 遭う（被災）/ 合う（一致）
- あらわす: 表す（表現）/ 現す（出現）/ 著す（執筆）
- きく: 聞く（音を聞く）/ 聴く（注意して聞く）/ 利く（効果がある）
- さわる: 触る（接触）/ 障る（害になる）

**出力**: 誤用と判定した漢字・正しい候補・使い分けの根拠

## なぜ L1/L2 で不可能か

- 「遭う」が正しいか「会う」が正しいかは、**文の意味（対象者・状況の種類）** に完全に依存する。「雨に遭う」は正しく「雨に会う」は誤りだが、「友人に会う」は正しく「友人に遭う」は誤り。
- 正規表現は「あう」という読みの出現を検出できても、どの漢字が適切かを判定できない。
- 形態素解析も読みを付与するが、複数の漢字候補のうちどれが文脈に合うかの意味的判断はできない。
- 文脈が複数文にまたがる場合（「災害が発生した。多くの人が〜に遭った」）は前文の情報が必要。

## 例

**positive（文脈不一致の指摘）**:

```
昨日の大雨で多くの市民が交通渋滞に会った。
```
→ 「交通渋滞」に「遭う」（好ましくない出来事）が適切 → 「遭った」への修正を提案。

**negative（文脈一致）**:

```
昨日の大雨で多くの市民が交通渋滞に遭った。
```
→ 「遭う」の使い方が文脈（被災・不本意な遭遇）に一致 → 指摘なし。

## 偽陽性・運用注意

- 「合う」は非常に広い意味を持ち（一致・調和・互い）、誤用判定が難しい場合がある。信頼度が低い場合は指摘を保留または info 扱いにする。
- 小説では語り口の特徴として意図的な誤字・旧用法を使う場合があるため、novel モードでは severity を info まで下げる運用を推奨。
- 同一の訓で複数の解釈が成立する場合（曖昧文）は指摘せず、確実に誤りと判断できる場合のみ報告する。
- 引用文・会話文内の誤用はスタイルの問題として扱い、指摘対象から外すことを検討する。
