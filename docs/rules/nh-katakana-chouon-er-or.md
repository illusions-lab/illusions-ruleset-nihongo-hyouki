# nh-katakana-chouon-er-or — 語末 -er/-or/-ar に相当する音は長音符「ー」で表記

- **ruleId**: `nh-katakana-chouon-er-or`
- **ガイドライン**: 日本語表記（日本エディタースクール）`nihongo-hyouki`
- **レベル**: L1（正規表現）
- **既定**: 有効 / severity = warning
- **適用モード**: 小説 / 公用文 / ブログ / 学術 / SNS（全モード）

## 意図

英語語末の -er/-or/-ar などに当たる音は、原則として長音符「ー」を用いて表記します。

## 出典

日本語表記（日本エディタースクール）第8章 外来語の表記 §3(2)④

原文要旨（p.38）:
> 英語の語末の -er, -or, -ar などに当たるものは、原則としてア列の長音とし音引「ー」を用いて書く。
> ただし、慣用で音引「ー」を省くものもある。
> （揺れがある例: エレベーター（エレベータ）、コンピューター（コンピュータ）、トランジスター（トランジスタ））

## 実装方針（ホワイトリスト方式）

書籍は「原則長音符を付ける」としつつ「慣用で省くものもある」「揺れがある」と両形を許容する例を
明示しているため、**ホワイトリスト方式**を採用しています。

- 英語 -er/-or/-ar 由来が明確で、かつ「ー無し形」が明らかに誤りとなる語を明示列挙する。
- 書籍が「揺れあり（両形許容）」と明示した語（コンピュータ、エレベータ、トランジスタ等）は
  ホワイトリストに**含めない**（誤りと断定できないため）。
- 英語 -er/-or/-ar 由来でない語（ドラマ、テーマ、ソナタ、パジャマ、オペラ、カメラ等）は
  最初から**対象外**（除外リストではなく、そもそも検出対象にしない）。
- 複合語途中の誤検出を防ぐため、語末確定（直後が非カタカナ）を先読みで保証する。

## 例

| 区分 | テキスト | 期待 |
| ---- | -------- | ---- |
| 正例 | `ギターの音が響いた。スキャナーを使う。プリンターで印刷する。` | 指摘なし |
| 誤例 | `ギタの音が響いた。スキャナを使う。` | 指摘あり |

### 指摘しない語（非対象）

以下は英語 -er/-or/-ar 由来でないか、書籍が両形を許容しているため、指摘しません。

| 語 | 理由 |
| -- | ---- |
| ドラマ、テーマ、ソナタ、パジャマ、オペラ | -er/-or/-ar 由来でない |
| カメラ、バナナ、サラダ、アイデア | -er/-or/-ar 由来でない |
| コンピュータ / エレベータ / トランジスタ | 書籍 p.38 で「揺れがある例」として両形を明示許容 |
| スリッパ、エンジニア | 書籍 p.38 で慣用省略例として明示 |

## 既知の制限

- ホワイトリストに含まれない語（コンピュータ・エレベータ等）はノーチェック。

- 新たに問題が報告された語はホワイトリストへの追加で対応する。
