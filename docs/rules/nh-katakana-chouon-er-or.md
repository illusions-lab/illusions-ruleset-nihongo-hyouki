# nh-katakana-chouon-er-or — 語末 -er/-or/-ar に相当する音は長音符「ー」で表記

- **ruleId**: `nh-katakana-chouon-er-or`
- **ガイドライン**: 日本語表記（日本エディタースクール）`nihongo-hyouki`
- **レベル**: L1（正規表現）
- **既定**: 有効 / severity = warning
- **適用モード**: 小説 / 公用文 / ブログ / 学術 / SNS（全モード）

## 意図

英語語末の -er/-or/-ar などに当たる音は、原則として長音符「ー」を用いて表記します。

## 出典

日本語表記（日本エディタースクール）第8章 外来語の表記 §3(2)④

## 例

| 区分 | テキスト | 期待 |
| ---- | -------- | ---- |
| 正例 | `ギターの音が響いた。マフラーを巻いた。` | 指摘なし |
| 誤例 | `ギタの音が響いた。マフラを巻いた。` | 指摘あり |

## 偽陽性リスク（低〜中）

本ルールは原文に明記された規約に基づき、偽陽性が起きにくいパターンのみを対象とする。
文脈によっては慣用として許容される場合があるため severity = warning とする。
