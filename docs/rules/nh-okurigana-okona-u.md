# nh-okurigana-okona-u — 「行なう」→「行う」（送り仮名の本則）

- **ruleId**: `nh-okurigana-okona-u`
- **ガイドライン**: 日本語表記（日本エディタースクール）`nihongo-hyouki`
- **レベル**: L1（正規表現）
- **既定**: 有効 / severity = warning
- **適用モード**: 小説 / 公用文 / ブログ / 学術 / SNS（全モード）

## 意図

「行なう」は旧規程の送り仮名です。「送り仮名の付け方」（1973年）の本則は「行う」です。

## 出典

日本語表記（日本エディタースクール）第7章 送り仮名の付け方 §3(許容→本則) p.32

## 例

| 区分 | テキスト | 期待 |
| ---- | -------- | ---- |
| 正例 | `検査を行う。` | 指摘なし |
| 誤例 | `検査を行なう。` | 指摘あり |

## 偽陽性リスク（低〜中）

本ルールは原文に明記された規約に基づき、偽陽性が起きにくいパターンのみを対象とする。
文脈によっては慣用として許容される場合があるため severity = warning とする。
