# nh-okurigana-kotowa-ru — 「断わる」→「断る」（送り仮名の本則）

- **ruleId**: `nh-okurigana-kotowa-ru`
- **ガイドライン**: 日本語表記（日本エディタースクール）`nihongo-hyouki`
- **レベル**: L1（正規表現）
- **既定**: 有効 / severity = warning
- **適用モード**: 小説 / 公用文 / ブログ / 学術 / SNS（全モード）

## 意図

「断わる」「賜わる」は旧形です。「送り仮名の付け方」（1973年）の本則は「断る」「賜る」です。

## 出典

日本語表記（日本エディタースクール）第7章 送り仮名の付け方 §2(3) 活用のある語の許容

## 例

| 区分 | テキスト | 期待 |
| ---- | -------- | ---- |
| 正例 | `依頼を断る。` | 指摘なし |
| 誤例 | `依頼を断わる。` | 指摘あり |

## 偽陽性リスク（低〜中）

本ルールは原文に明記された規約に基づき、偽陽性が起きにくいパターンのみを対象とする。
文脈によっては慣用として許容される場合があるため severity = warning とする。
