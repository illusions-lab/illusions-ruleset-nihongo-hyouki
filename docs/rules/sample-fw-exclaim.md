# sample-fw-exclaim — 全角『！』の検出

> 校正目録（ルールごとのドキュメント）テンプレート。新しいルールを追加したら、この形式で
> `docs/<ruleId>.md` を1つ作る。`manifest.json` の `docs` 例とテストを一致させること。

- **ruleId**: `sample-fw-exclaim`
- **ガイドライン**: わたしの規約 (`my-guideline`)
- **レベル**: L1（正規表現）
- **既定**: 有効 / severity = info
- **適用モード**: 小説 (`novel`) / ブログ (`blog`) — これらのモードに切り替えると自動で有効化される

## 意図

和文の本文では感嘆符を半角『!』に統一する。全角『！』を検出して半角への置換を提案する。

## 出典

サンプル規約 §1

## 例

| 区分 | テキスト | 期待 |
| --- | --- | --- |
| 正例 | `すごい!` | 指摘なし |
| 誤例 | `すごい！` | `！` を `!` に置換提案 |

## 実装メモ

- `ctx.toolkit.regexReplace` で `/！/` を検出。
- 依存（辞典など）は不要。辞典依存ルールは `manifest.rules[].requires` に
  `{ "kind": "dict", "dictId": "genji" }` を宣言する（未DL時は自動無効化＋警告）。
