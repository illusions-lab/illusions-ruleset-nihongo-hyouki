# nh-hojo-verb-l2 — 補助動詞「ていく・てくる」は仮名書き（形態素解析）

- **ruleId**: `nh-hojo-verb-l2`
- **ガイドライン**: 日本語表記（日本エディタースクール）`nihongo-hyouki`
- **レベル**: L2（形態素解析）
- **既定**: 有効 / severity = warning
- **適用モード**: 小説 / 公用文 / ブログ / 学術 / SNS（全モード）

## 意図

補助動詞として用いる「行く」「来る」は、仮名で「いく」「くる」と書きます。
接続助詞「て」の直後に動詞が来る文脈を形態素解析で確認し、補助動詞用法のみを対象とします。

### L1 ルール（nh-kana-auxiliary-verb）との役割分担

`nh-kana-auxiliary-verb`（L1）は正規表現で「てしまう」「てみる」「ておく」「てあげる」等を検出します。
しかし「行く」「来る」は本動詞としても頻出するため、正規表現では本動詞と補助動詞の区別ができず L1 の対象外です。
本ルール（L2）は形態素解析の接続助詞境界を利用することで、「行く」「来る」を安全に検出します。
両ルールは互いに補完し、重複指摘はありません。

## 出典

日本語表記（日本エディタースクール）第3章 漢字の用い方 §1 漢字と仮名の使用

## 例

| 区分 | テキスト | 期待 |
| ---- | -------- | ---- |
| 正例 | `プロジェクトが進んでいく。` | 指摘なし |
| 正例 | `春が近づいてくる。` | 指摘なし |
| 誤例 | `プロジェクトが進んで行く。` | 指摘あり（行く → いく） |
| 誤例 | `春が近づいて来る。` | 指摘あり（来る → くる） |

## 偽陽性リスク（低）

- 直前トークンが接続助詞「て」の場合のみ検出するため、本動詞用法（「図書館へ行く」「春が来る」）は除外される。
- surface が平仮名のみで構成されるトークンはスキップする。
- 「で」は格助詞（電車で行く）と接続助詞の区別が形態素解析でも困難なため対象外とする。
