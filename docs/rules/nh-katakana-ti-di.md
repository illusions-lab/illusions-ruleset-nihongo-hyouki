# nh-katakana-ti-di — 「ティ・ディ」（慣用を除く第1表の仮名）

- **ruleId**: `nh-katakana-ti-di`
- **ガイドライン**: 日本語表記（日本エディタースクール）`nihongo-hyouki`
- **レベル**: L1（正規表現）
- **既定**: 有効 / severity = warning
- **適用モード**: 小説 / 公用文 / ブログ / 学術 / SNS（全モード）

## 意図

原音の「ティ」「ディ」は、慣用を除き「ティ」「ディ」と書きます（「チ」「ジ」ではありません）。

## 出典

日本語表記（日本エディタースクール）第8章 外来語の表記 §3(3)④

## 例

| 区分 | テキスト | 期待 |
| ---- | -------- | ---- |
| 正例 | `ティーカップを持つ。ディスプレイを見る。` | 指摘なし |
| 誤例 | `ボランテイアに参加したパーテイーの記録。` | 指摘あり |

## 偽陽性リスク（低〜中）

本ルールは原文に明記された規約に基づき、偽陽性が起きにくいパターンのみを対象とする。
文脈によっては慣用として許容される場合があるため severity = warning とする。
