# nh-kana-auxiliary-verb — 補助動詞は仮名書き

- **ruleId**: `nh-kana-auxiliary-verb`
- **ガイドライン**: 日本語表記（日本エディタースクール）`nihongo-hyouki`
- **レベル**: L1（正規表現）
- **既定**: 有効 / severity = warning
- **適用モード**: 小説 / 公用文 / ブログ / 学術 / SNS（全モード）

## 意図

「してあげる」「していただく」「してみる」「してしまう」など、補助動詞として使う「あげる・いただく・みる・しまう」等は仮名で書きます。

## 出典

日本語表記（日本エディタースクール）第3章 漢字の用い方 §1 漢字と仮名の使用

## 例

| 区分 | テキスト | 期待 |
| ---- | -------- | ---- |
| 正例 | `本を読んでみる。宿題をやってしまう。` | 指摘なし |
| 誤例 | `本を読んで見る。` | 指摘あり |

## 偽陽性リスク（低〜中）

本ルールは原文に明記された規約に基づき、偽陽性が起きにくいパターンのみを対象とする。
文脈によっては慣用として許容される場合があるため severity = warning とする。
