# nh-katakana-fa-fi-fe-fo — 「ファ・フィ・フェ・フォ」（第1表の仮名）

- **ruleId**: `nh-katakana-fa-fi-fe-fo`
- **ガイドライン**: 日本語表記（日本エディタースクール）`nihongo-hyouki`
- **レベル**: L1（正規表現）
- **既定**: 有効 / severity = warning
- **適用モード**: 小説 / 公用文 / ブログ / 学術 / SNS（全モード）

## 意図

原音の「ファ・フィ・フェ・フォ」は、慣用を除き「ファ・フィ・フェ・フォ」と書きます（「ハ・ヒ・ヘ・ホ」ではありません）。

## 出典

日本語表記（日本エディタースクール）第8章 外来語の表記 §3(3)⑤ p.39

| 区分 | テキスト | 期待 |
| ---- | -------- | ---- |
| 正例 | `ファイルを開く。フィルムを現像する。` | 指摘なし |
| 誤例 | `フアイルを開く。` | 指摘あり |
