# nh-katakana-she-je — 「シェ」「ジェ」（第1表の仮名）

- **ruleId**: `nh-katakana-she-je`
- **ガイドライン**: 日本語表記（日本エディタースクール）`nihongo-hyouki`
- **レベル**: L1（正規表現）
- **既定**: 有効 / severity = warning
- **適用モード**: 小説 / 公用文 / ブログ / 学術 / SNS（全モード）

## 意図

原音の「シェ」「ジェ」は、慣用を除きカタカナで「シェ」「ジェ」と書きます（「セ」「ゼ」ではありません）。

## 出典

日本語表記（日本エディタースクール）第8章 外来語の表記 §3(3)① p.39

## 例

| 区分 | テキスト | 期待 |
| ---- | -------- | ---- |
| 正例 | `シェードを下げる。ジェットエンジンの音。` | 指摘なし |
| 誤例 | `シエードを下げる。` | 指摘あり |

## 偽陽性リスク（低〜中）

本ルールは原文に明記された規約に基づき、偽陽性が起きにくいパターンのみを対象とする。
文脈によっては慣用として許容される場合があるため severity = warning とする。
