# nh-katakana-di-du — 外来語の「ヂ」「ヅ」使用禁止

- **ruleId**: `nh-katakana-di-du`
- **ガイドライン**: 日本語表記（日本エディタースクール）`nihongo-hyouki`
- **レベル**: L1（正規表現）
- **既定**: 有効 / severity = warning
- **適用モード**: 小説 / 公用文 / ブログ / 学術 / SNS（全モード）

## 意図

外来語表記で「ヂ」「ヅ」（固有名詞を除く）は使用しません。「ジ」「ズ」と表記します。

## 出典

日本語表記（日本エディタースクール）第8章 外来語の表記 §3(1)

## 例

| 区分 | テキスト | 期待 |
| ---- | -------- | ---- |
| 正例 | `ジャズの演奏を聴いた。` | 指摘なし |
| 誤例 | `ヅカダンサーの公演があった。` | 指摘あり |

## 偽陽性リスク（低〜中）

本ルールは原文に明記された規約に基づき、偽陽性が起きにくいパターンのみを対象とする。
文脈によっては慣用として許容される場合があるため severity = warning とする。
