# nh-kana-formal-noun — 形式名詞「こと・もの・ところ・わけ」は仮名書き

- **ruleId**: `nh-kana-formal-noun`
- **ガイドライン**: 日本語表記（日本エディタースクール）`nihongo-hyouki`
- **レベル**: L2（形態素解析）
- **既定**: 有効 / severity = warning
- **適用モード**: 小説 / 公用文 / ブログ / 学術 / SNS（全モード）

## 意図

形式名詞として用いる「事・物・所・訳」は、仮名で「こと・もの・ところ・わけ」と書きます。
「一般名詞」（事件・物語・所属・訳語など）は対象外です。

## 出典

日本語表記（日本エディタースクール）第3章 漢字の用い方 §1 p.9

## 例

| 区分 | テキスト | 期待 |
| ---- | -------- | ---- |
| 正例 | `勉強することが大切だ。` | 指摘なし |
| 誤例 | `勉強する事が大切だ。` | 指摘あり |

## 偽陽性リスク（低）

kuromoji が品詞細分類「非自立」でタグ付けする「事/物/所/訳」のみを対象とするため、
一般名詞用法（事件、物語、所属、訳語）との混同リスクは低い。
