# nh-katakana-sokuon — 促音「ツ」の大書き禁止（外来語）

- **ruleId**: `nh-katakana-sokuon`
- **ガイドライン**: 日本語表記（日本エディタースクール）`nihongo-hyouki`
- **レベル**: L1（正規表現）
- **既定**: 有効 / severity = warning
- **適用モード**: 小説 / 公用文 / ブログ / 学術 / SNS（全モード）

## 意図

外来語表記の促音（つまる音）は小書きの「ッ」を用います。大書きの「ツ」は不可です。

## 出典

日本語表記（日本エディタースクール）第8章 外来語の表記 §3(2)①

## 例

| 区分 | テキスト | 期待 |
| ---- | -------- | ---- |
| 正例 | `バッグを持った。コップが割れた。` | 指摘なし |
| 誤例 | `バツグを持った。コツプが割れた。` | 指摘あり |

## 既知の制限

### 除外語を含む複合固有名詞

除外リスト（カツラ、マツダ、ウォルツ等）は**単独の語**として登場する場合に適用されます。
これらの語が複合固有名詞の一部として含まれる場合（例: カツラギ山、カツラ川）は、
複合語全体を抽出して除外リストと照合するため、除外リストに「カツラ」があっても
「カツラギ」は除外リストに含まれておらず、誤検出することがあります。

**回避策**: 除外リストに複合形も追加する（`WORD_EXCLUSIONS.add("カツラギ")`）か、
固有名詞は lint 対象から除外する上位の設定で対応してください。

現時点では固有名詞複合語の網羅的な除外は行っておらず、実運用で問題が確認された語を
都度追加する方針とします。

## 偽陽性リスク（低〜中）

本ルールは原文に明記された規約に基づき、偽陽性が起きにくいパターンのみを対象とする。
文脈によっては慣用として許容される場合があるため severity = warning とする。
