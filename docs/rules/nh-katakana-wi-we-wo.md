# nh-katakana-wi-we-wo — 外来語の旧仮名（ヰ・ヱ・ヲ）使用禁止

- **ruleId**: `nh-katakana-wi-we-wo`
- **ガイドライン**: 日本語表記（日本エディタースクール）`nihongo-hyouki`
- **レベル**: L1（正規表現）
- **既定**: 有効 / severity = warning
- **適用モード**: 小説 / 公用文 / ブログ / 学術 / SNS（全モード）

## 意図

「外来語の表記」（1991年内閣告示）§3(1) は、固有名詞などを除き、
旧仮名遣いで使用する「ヰ」「ヱ」「ヲ」は外来語表記に使用しないと定める。
これらの旧仮名が使われている箇所を検出し、現代的な表記への修正を提案する。

## 出典

日本語表記（日本エディタースクール）第8章 外来語の表記 §3(1)

## 例

| 区分 | テキスト | 期待 |
| ---- | -------- | ---- |
| 正例 | `ウィスキーを飲む。` | 指摘なし |
| 誤例 | `スヰフトの作品を読んだ。` | `ヰ` を指摘 |

## 偽陽性リスク（低）

- 「ヰ」「ヱ」「ヲ」は現代テキストにほぼ現れないため偽陽性リスクは極めて低い。
- 固有名詞（ヱビス等）は対象になりうるが、severity=warning のため手動確認を促す。

## 実装メモ

- 三つの旧仮名をそれぞれ個別パターンで検出。修正候補は「イ」「エ」「オ」を提案（文脈によっては「ウィ」等が適切）。
