# nh-l3-hyoki-yure-kanji-kana — 同一語の漢字／かな表記ゆれ（文書内統一）

- **ガイドライン**: 日本語表記（日本エディタースクール）`nihongo-hyouki`
- **レベル**: L3（LLM支援・文書全体分析）
- **想定 severity**: warning
- **適用モード案**: 小説 (`novel`) / 公用文 (`official`) / ブログ (`blog`) / 学術 (`academic`) / SNS (`sns`)

## 意図

代名詞・副詞・接続詞・連体詞・形式名詞・補助動詞などは、漢字で書くか仮名で書くかに揺れが生じやすい。
同一原稿内でこの揺れが不統一なままになることは「あまり体裁のよいことではない」と書籍は明示している。
文書全体で同一語の表記を洗い出して不統一を検出するには、語の意味的同一性の判断が必要であり、LLM による分析が適している。

## 書籍根拠

日本語表記（日本エディタースクール）序文 §3

現代表記にはある程度の幅や揺れが伴う。原稿内で表記の揺れが不統一のままになることは
体裁上の問題につながるとして、書籍では表記の揺れを整理する考え方が解説されている。

第3章「漢字の用い方」§3 代名詞・副詞・接続詞などの扱い

代名詞・副詞・接続詞・連体詞・接頭語などでは、漢字を使う場合と仮名書きにする場合があるため、
一定の方針を決めて不統一を避けるよう整理することが求められる。
熟字訓・当て字・動植物名でも漢字か仮名かの揺れが生じやすいと書籍は指摘している。

Word 校正支援機能の解説欄でも「表記の揺れ」の点検項目として「揺らぎ（漢字/仮名）」が
明示されている。

## 動作原理

**LLM への入力**: 文書全体のテキスト

**判断基準**:
1. 文書内に出現する語を形態素解析等で列挙し、同一語（読みが一致し意味が等価）の異表記ペアを収集する（例: 「事」と「こと」、「時」と「とき」、「出来る」と「できる」）。
2. 一方の表記が多数派であれば、少数派の出現箇所を逸脱として報告する。
3. 意図的な使い分け（実質名詞としての漢字表記 vs 形式名詞としての仮名表記）は除外する。

**出力**: 揺れのある語ペア・各出現箇所・多数派表記への統一案

## なぜ L1/L2 で不可能か

- 「こと」と「事」が同一文脈で等価かどうかは **文中の用法（形式名詞か実質名詞か）** に依存する。
  - 「驚くべきことだ」の「こと」（形式名詞） vs「大切な事柄」の「事」（実質名詞）は別語として扱うべき。
  - この区別は形態素解析だけでは不十分で、文の意味的分析が必要。
- 「文書内の多数派を基準にする」という判断自体が文書全体を参照するため、単文スコープの L1/L2 では実装不可能。

## 例

**positive（不統一の指摘）**:

```
会議の結果として、できることはすべて行う。次回は出来るだけ早く報告する。
```
→ 「できる」と「出来る」が混在 → 多数派（できる）に統一を提案。

**negative（一貫した表記）**:

```
できることはすべて行う。できるだけ早く報告する。
```
→ 「できる」で統一されているため指摘なし。

## 偽陽性・運用注意

- 意図的な対比のために表記を使い分けている場合（文学的効果・引用等）は誤検出の可能性がある。
- 固有名詞・専門用語・引用文は評価対象から外す。
- 許容表記が複数ある語（「奥さん」「奥様」等）は同義ではなく語義・敬意が異なるため、単純な揺れ扱いをしない。
- 文書が短すぎる（10文未満）場合は多数派判定が不安定なため適用を保留することを推奨。
